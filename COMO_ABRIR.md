# 🚀 Como Abrir a Landing Page

## Opção 1: Arquivo Local (Mais Fácil)
1. Baixe o arquivo: `landing-page-masterclass.html`
2. Abra direto no navegador (desktop ou celular)
3. Funciona 100% offline

## Opção 2: Copiar Arquivo HTML
Se estiver no seu computador pessoal:

```bash
# Copie o arquivo para sua máquina
# Depois abra direto no navegador
file:///seu-caminho/landing-page-masterclass.html
```

## Opção 3: Deploy Online (Gratuito)
Use Netlify Drop ou Vercel:
1. Acesse: https://app.netlify.com/drop
2. Arraste `landing-page-masterclass.html` 
3. Pronto! Compartilhe o link com qualquer um
